# QQ Farm Vision Bot

基于 OpenCV 视觉识别的 QQ 经典农场（微信小程序/QQ 小程序）自动化工具。纯本地运行，不依赖游戏接口，零封号风险。

**项目来源**：基于 [Z7ANN/qq-farm-auto](https://github.com/Z7ANN/qq-farm-auto) 修改，适配个人账号的土地情况并优化窗口查找逻辑。同时参考了 [megumiss/qq-farm-copilot](https://github.com/megumiss/qq-farm-copilot) 的实现思路。

**Gitee 镜像**：[luckytiger12138/qq-farm](https://gitee.com/luckytiger12138/qq-farm)

## 工作原理

```
截屏 (mss) → OpenCV 多尺度模板匹配 → 场景识别 → 策略决策 → PostMessageW 后台点击 / pyautogui 前台点击 → 循环
```

通过截取游戏窗口画面，用 OpenCV 模板匹配识别按钮、图标、作物状态，再按优先级自动执行操作。不读取内存、不修改数据包、不调用游戏 API。支持后台运行模式（PostMessageW），操作时鼠标不被抢占，可正常使用电脑。

## 环境要求

- Windows 10/11
- Python 3.10+
- PC 端微信或 PC 版 QQ，打开 QQ 经典农场小程序

## 安装

### 方式一：一键安装（推荐）

Windows 用户直接双击运行 `setup.bat`，自动完成依赖安装和桌面快捷方式创建。

### 方式二：手动安装

```bash
git clone https://gitee.com/luckytiger12138/qq-farm.git
cd qq-farm
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt
```

## 快速开始

### 1. 采集模板（首次使用必须）

打开微信或 QQ 中的 QQ 经典农场小程序，然后运行：

```bash
python tools/template_collector.py
```

**操作方式：**
- 鼠标左键逐点点击绘制多边形轮廓
- 按 `Enter` 完成多边形
- 右键撤销最后一个点
- 按 `R` 重新截屏
- 按 `Q` 退出

保存的 PNG 文件带 alpha 透明通道，背景自动剔除，匹配更精准。

> **提示：** 也可在 GUI 模板管理面板中直接截屏采集，支持多边形框选。

### 2. 调节阈值（推荐）

启动 `main.py` 后，在模板管理面板中：

1. 点击「截取窗口」获取当前游戏画面
2. 点击「批量测试同类别」查看同类别所有模板的匹配情况
3. 对匹配不准确的模板单独调节阈值
4. 阈值自动保存到 `templates/thresholds.json`，下次启动自动加载

**阈值优先级：** 单模板阈值 > 类别阈值 > 内置默认阈值

### 3. 配置游戏路径（可选但推荐）

启动 `main.py` 后，在设置面板 → 其他 → 游戏路径 中：
1. 点击「浏览...」按钮
2. 选择 QQ 农场小程序的快捷方式（`.lnk` 文件）
3. 自动保存

配置后，Bot 启动时若未检测到游戏窗口会自动启动游戏。

### 4. 启动

**方式一：双击桌面快捷方式**（推荐）

安装完成后会自动创建 "QQ 农场助手" 桌面快捷方式，双击即可启动。

**方式二：命令行启动**

```bash
python main.py
```

### 5. 配置

GUI 左侧导航栏包含四个页面：

- **状态总览**：运行状态渐变卡片、已运行/下次检查时间、6 个彩色操作统计卡片（收获/播种/浇水/除草/除虫/施肥）、实时截图预览
- **参数设置**：
  - 种植：等级、策略（自动最优/手动指定）、推荐作物、指定作物
  - 功能开关：12 项功能一键启停（收获/播种/施肥/买种/浇水/除草/除虫/出售/偷菜/帮忙/任务/扩建）
  - 出售策略：出售模式（批量全部/选择性）、选择性作物多选
  - 其他：窗口关键词、游戏路径、检查间隔、运行模式（前台/后台）
- **模板管理**：
  - 模板卡片列表（搜索、按类别/名称筛选、排序）
  - 点击模板卡片展开右侧详情面板
  - 详情面板：预览图、信息、阈值调节、匹配测试、批量测试同类别、编辑操作（重命名/替换图片/截屏替换/启用禁用/删除）
  - 工具栏：截屏采集、导入、刷新、类别阈值设置
- **运行日志**：深色终端风格日志，支持 INFO/WARN/ERROR 过滤、一键复制、一键清空

所有设置修改后实时生效，无需手动保存。

### 6. 运行

点击「开始」按钮，程序会：

1. 自动找到 QQ 经典农场窗口（未找到时通过快捷方式自动启动）
2. 调整窗口大小并定位到屏幕左下角（预留任务栏）
3. 清屏：点击天空区域关闭残留弹窗
4. 循环执行：截屏 → 场景识别 → 按优先级执行操作
5. 播种后按设置间隔定期检查维护（除虫/除草/浇水/任务等）
6. 检测到升级弹窗自动更新等级，重新计算最优作物
7. 检测到异地登录自动关闭游戏，等待 3 分钟后重启
8. 运行中游戏窗口意外关闭自动重启

热键：
- `F9` — 暂停 / 恢复
- `F10` — 强制停止

### 运行模式

| 模式 | 说明 |
|------|------|
| **前台** (foreground) | 使用 pyautogui 模拟鼠标，操作时不能使用电脑 |
| **后台** (background) | 使用 PostMessageW 发送消息，不抢占鼠标，窗口可被遮挡，可正常使用电脑 |

后台模式支持所有操作（点击、拖拽播种、拖拽施肥等），推荐日常使用。

## 模板命名规范

| 前缀 | 类别 | 示例 |
|------|------|------|
| `btn_` | button | `btn_harvest.png` |
| `bth_` | button（特殊按钮如施肥） | `bth_feiliao_pt.png` |
| `icon_` | status_icon | `icon_mature.png` |
| `crop_` | crop | `crop_mature.png` |
| `seed_` | seed（播种列表） | `seed_小麦.png` |
| `shop_` | shop（商店卡片） | `shop_小麦.png` |
| `land_` | land | `land_empty.png`, `land_金1.png` |
| `ui_` | ui_element | `ui_next_time.png` |

前缀与 `TEMPLATE_CATEGORIES` 字典映射决定模板分类。新增前缀需同时更新 `cv_detector.py` 中的 `TEMPLATE_CATEGORIES`。

## 常见问题

**Q: 找不到游戏窗口？**

1. 确保 PC 端微信或 QQ 已打开 QQ 经典农场小程序
2. 可在设置 → 其他 → 游戏路径 中配置快捷方式，Bot 会自动启动游戏
3. 窗口关键词默认是 "QQ 经典农场"，如果实际窗口标题不同，请在设置中修改

**Q: 模板匹配不准确？**

1. 模板需要在自己的设备上采集，不同分辨率、DPI 缩放会导致不匹配
2. 在模板管理面板中调节单模板阈值或类别默认阈值
3. 使用「批量测试同类别」功能查看同类别所有模板的匹配情况
4. 对于不规则形状，使用多边形框选采集，保存带 alpha 蒙版的 PNG

**Q: 同一块地被识别多次？**

系统已内置按类别中心点距离去重逻辑，同一位置只保留置信度最高的一个。如果仍有重复，可在模板管理中禁用多余的同类模板。

**Q: 游戏意外关闭怎么办？**

Bot 内置实时窗口监控（每 5 秒检测一次），检测到游戏窗口关闭会自动重启游戏并继续执行任务。

**Q: 异地登录怎么处理？**

检测到异地登录弹窗（`ui_remote_login` 或 `ui_next_time` 模板）时，Bot 会自动关闭游戏窗口并重新打开，无需人工干预。

**Q: 如何添加新功能？**

在 `core/strategies/` 下新建策略模块，继承 `BaseStrategy`，在 `bot_engine.py` 中注册并按优先级编排。
