# QQ Farm Vision Bot

基于 OpenCV 视觉识别的 QQ 经典农场自动化工具。纯本地运行，不读取内存、不修改数据包，零封号风险。

> **项目来源**：基于 [Z7ANN/qq-farm-auto](https://github.com/Z7ANN/qq-farm-auto) 修改，参考 [megumiss/qq-farm-copilot](https://github.com/megumiss/qq-farm-copilot) 思路。

**仓库**：[GitHub](https://github.com/LuckyTiger12138/QQ_Farm) · [Gitee](https://gitee.com/luckytiger12138/qq-farm)

> Gitee 发行版限制 100MB，可执行文件请从 [GitHub Releases](https://github.com/LuckyTiger12138/QQ_Farm/releases) 下载。

## 环境要求

- Windows 10/11
- Python 3.10+
- **显示器建议 16:9 比例**（部分 16:10 笔记本或非标准比例显示器可能存在坐标精度丢失问题）
- PC 端微信或 QQ，打开 QQ 经典农场小程序

## 快速开始

### 1. 安装

**一键安装**：双击 `setup.bat`，自动安装依赖并创建桌面快捷方式。

**手动安装**：
```bash
git clone https://gitee.com/luckytiger12138/qq-farm.git
cd qq-farm
# 国内用户推荐使用镜像加速
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt
```

### 2. 启动程序
```bash
python main.py
```
双击桌面快捷方式同样可以启动。

### 3. 模板检查与采集（首次使用必须）

启动后进入 **「模板管理」** 面板：

1. **检查模板完整性**：根据当前玩家等级和想种植的作物，确认是否有对应的 `seed_` 和 `shop_` 模板
   - 缺少模板时，点击「截屏采集」用多边形框选采集
   - 种子模板也可批量导入：`python tools/import_seeds.py`

2. **测试匹配准确性**：
   - 点击「截取窗口」获取当前游戏画面
   - 点击「批量测试同类别」查看同类别所有模板的匹配情况
   - 默认阈值下能正确匹配即可，无需调整
   - 匹配不准确的模板，单独调节阈值或重新采集

3. **调节阈值**（可选）：
   - 单模板阈值 > 类别阈值 > 内置默认阈值
   - 阈值自动保存到 `templates/thresholds.json`

> **提示**：模板需要在自己的设备上采集，不同分辨率/DPI 会导致不匹配。不规则形状（如按钮图标）建议使用多边形框选，保存带 alpha 通道的 PNG。

### 4. 配置游戏路径（可选）

设置面板 → 游戏路径 → 选择 QQ 农场快捷方式。配置后 Bot 会自动启动未打开的游戏。

### 5. 运行

点击「开始」按钮，程序会：
1. 自动找到或启动游戏窗口
2. 调整窗口大小并定位
3. 循环执行：截屏 → 场景识别 → 按优先级执行操作

**热键**：F9 暂停/恢复，F10 停止

### 6. 运行模式

| 模式 | 说明 |
|------|------|
| **后台** (推荐) | PostMessageW 消息模拟，不抢占鼠标，窗口可遮挡 |
| **前台** | pyautogui 模拟鼠标，操作时不能使用电脑 |

### 7. 多实例支持（同时挂多个号）

程序支持同时运行多个农场实例（不同 QQ 号），每个实例拥有独立的配置、日志和截图。

#### 7.1 窗口位置管理

多实例运行时，建议将不同游戏窗口分散到屏幕不同位置，避免重叠：

1. 打开设置面板，找到「📍 窗口位置」选项
2. 为不同实例设置不同的位置：
   - 实例 1：🔽 左下角（默认）
   - 实例 2：🔼 左上角
   - 实例 3：🔽 右下角
   - 实例 4：🔼 右上角
   - 或者选择 ⊙ 居中

#### 7.2 窗口选择规则

每个实例需要绑定不同的游戏窗口：

1. 打开设置面板，找到「🎯 选择窗口」下拉框
2. 配置规则：
   - **自动**：按平台优先选择（适合单实例）
   - **#1 [QQ]QQ 经典农场**：选择第 1 个窗口
   - **#2 [微信]QQ 经典农场**：选择第 2 个窗口
   - 以此类推...

> **提示**：程序会自动锁定已选定的窗口（通过 hwnd 绑定），即使窗口位置发生变化也不会选错。

#### 7.3 操作步骤

1. 打开两个 QQ/微信窗口，分别登录 QQ 经典农场
2. 在程序右侧实例栏，点击「+」创建新实例
3. 选中新实例，在设置面板中：
   - 配置不同的「窗口位置」（如右上角）
   - 选择对应的游戏窗口（如 #2）
4. 返回实例栏，分别点击每个实例的「▶️」启动按钮
5. 每个实例会独立运行，互不干扰

#### 7.4 实例管理

| 操作 | 说明 |
|------|------|
| **启动/停止** | 实例栏每个实例独立控制 |
| **全部启动/全部停止** | 一键控制所有实例 |
| **克隆实例** | 快速复制配置到新实例 |
| **重命名** | 方便识别不同账号 |
| **删除** | 移除不需要的实例（至少保留 1 个） |

> **注意**：多实例同时运行时，截图预览区只会显示**当前选中实例**的画面。其他实例在后台独立运行，不会干扰预览。

## Web 端控制（可选）

设置面板 → Web 服务 中启用后，访问 `http://localhost:8080`（端口可自定义）。

**功能**：实时截图（2 秒刷新）、状态显示、启停控制、完整配置编辑、最近 50 条日志。

## 常见问题

**找不到游戏窗口？** 确保小程序已打开；或配置游戏快捷方式让 Bot 自动启动。

**模板匹配不准确？** 重新采集模板；调节阈值；不规则形状用多边形框选。

**同一块地被识别多次？** 系统已内置去重逻辑。仍有重复可在模板管理中禁用多余同类模板。

**游戏意外关闭？** 每 5 秒自动检测，关闭后自动重启。

**异地登录？** 自动关闭游戏窗口并重新打开，无需人工干预。
