# QQ Farm Vision Bot

基于 OpenCV 视觉识别的 QQ 经典农场（微信小程序/QQ 小程序）自动化工具。纯本地运行，不依赖游戏接口，零封号风险。

**项目来源**：基于 [Z7ANN/qq-farm-auto](https://github.com/Z7ANN/qq-farm-auto) 修改，适配个人账号的土地情况并优化窗口查找逻辑。

**Gitee 镜像**：[luckytiger12138/qq-farm](https://gitee.com/luckytiger12138/qq-farm)

## 工作原理

```
截屏 (mss) → OpenCV 多尺度模板匹配 → 场景识别 → 策略决策 → pyautogui 模拟点击 → 循环
```

通过截取游戏窗口画面，用 OpenCV 模板匹配识别按钮、图标、作物状态，再按优先级自动执行操作。不读取内存、不修改数据包、不调用游戏 API。

## 环境要求

- Windows 10/11
- Python 3.10+
- PC 端微信或 PC 版 QQ，打开 QQ 经典农场小程序

## 安装

### 方式一：一键安装（推荐）

Windows 用户直接双击运行 `setup.bat`，自动完成依赖安装和桌面快捷方式创建。

### 方式二：手动安装

```bash
git clone https://gitee.com/luckytiger12138/qq-farm.git
cd qq-farm
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt
```

## 快速开始

### 1. 采集模板（首次使用必须）

打开微信或 QQ 中的 QQ 经典农场小程序，然后运行：

```bash
python tools/template_collector.py
```

操作方式：鼠标左键拖拽框选 → 按 `S` 保存 → 按 `R` 重新截屏 → 按 `Q` 退出

种子模板可通过导入工具批量生成：

```bash
python tools/import_seeds.py
```

**导入工具功能说明：**
- 自动从 GitHub 仓库获取最新种子图片
- 同时生成 `seed_`（播种列表）和 `shop_`（商店卡片）两种模板
- 与本地模板对比，跳过已存在的文件，避免重复下载
- 支持代理访问（默认使用 Clash 代理 `http://127.0.0.1:7890`）
- 自动处理网络错误，支持重试机制

> **提示：** 也可在 GUI 模板管理面板中直接截屏采集或导入图片，无需命令行操作。

### 2. 配置游戏路径（可选但推荐）

启动 `main.py` 后，在设置面板 → 其他 → 游戏路径 中：
1. 点击「浏览...」按钮
2. 选择 QQ 农场小程序的快捷方式（`.lnk` 文件）
3. 自动保存

配置后，Bot 启动时若未检测到游戏窗口会自动启动游戏。

### 3. 启动

**方式一：双击桌面快捷方式**（推荐）

安装完成后会自动创建 "QQ 农场助手" 桌面快捷方式，双击即可启动。

**方式二：命令行启动**

```bash
python main.py
```

### 4. 配置

GUI 左侧导航栏包含五个页面：

- **状态**：运行状态、操作统计、实时截图预览
- **模板管理**：
  - 模板列表（搜索、按类别/名称筛选、排序）
  - 点击模板卡片展开右侧详情面板
  - 详情面板：预览图、信息、阈值调节、匹配测试、编辑操作
  - 工具栏：截屏采集、导入、刷新、类别阈值设置
- **设置**：
  - 等级、种植策略、功能开关、买种开关
  - 检查间隔（农场 / 好友）
  - 游戏路径、窗口关键词
- **出售**：出售模式（批量全部 / 选择性），选择性模式可勾选要出售的作物
- **日志**：实时操作日志

所有设置修改后实时生效，无需手动保存。

### 5. 运行

点击「开始」按钮，程序会：

1. 自动找到 QQ 经典农场窗口（未找到时通过快捷方式自动启动）
2. 调整窗口大小并定位到屏幕左下角（预留任务栏）
3. 清屏：点击天空区域关闭残留弹窗
4. 循环执行：截屏 → 场景识别 → 按优先级执行操作
5. 播种后按设置间隔定期检查维护（除虫/除草/浇水/任务等）
6. 检测到升级弹窗自动更新等级，重新计算最优作物

热键：
- `F9` — 暂停 / 恢复
- `F10` — 强制停止

## 常见问题

**Q: 找不到游戏窗口？**

1. 确保 PC 端微信或 QQ 已打开 QQ 经典农场小程序
2. 可在设置 → 其他 → 游戏路径 中配置快捷方式，Bot 会自动启动游戏
3. 窗口关键词默认是 "QQ 经典农场"，如果实际窗口标题不同，请在设置中修改

**Q: 模板匹配不准确？**

模板需要在自己的设备上采集。不同分辨率、DPI 缩放会导致模板不匹配。可在模板管理面板中调节单模板阈值或类别默认阈值。

**Q: 如何添加新功能？**

在 `core/strategies/` 下新建策略模块，继承 `BaseStrategy`，在 `bot_engine.py` 中注册并按优先级编排。
